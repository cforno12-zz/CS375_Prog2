How to run:
    > make
    > ./BestFirstSearch <input.txt> <output.txt>

Data Structures:
     For the branch & bound algorithm, a priority queue is used to determine the next node to visit. There are also vectors(arrays) used.

Classes used:
        Item:
          class used to hold profit and weight of a single item
        Node:
          class used to hold information about a node in the decision tree
